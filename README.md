# Tokyo Night - Tweaks

This repository is used for tweaks to the amazing [Tokyo Night](https://github.com/folke/tokyonight.nvim) theme.

The aim is to have both dark and light (night and day) variants of the Tokyo Night theme for use with a terminal emulator and TUI applications.

## Themes in this repository

- [tokyonight-day.yazi](tokyonight-day.yazi/README.md): Light theme for [yazi](https://github.com/sxyazi/yazi).
- [tokyonight-night.yazi](tokyonight-night.yazi/README.md): Dark theme for [yazi](https://github.com/sxyazi/yazi).
